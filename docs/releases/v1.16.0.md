This is a minor release including the following changes:

- Add support for Kubernetes 1.34
- Update Kyverno to version 1.16.0

### Process

#### Gatekeeper

To upgrade this package from `v1.15.1` to `v1.16.0`, you need to download this new version, then apply the `kustomize` project. No further action is required.

```bash
kustomize build katalog/gatekeeper | kubectl apply -f -
```
#### Kyverno

To upgrade this package from `v1.15.1` to `v1.16.0`, you need to download this new version, then apply the `kustomize` project. No further action is required.
